## 跑路前一般有什么征兆？

很多时候，老板想卷款走人是有迹可循的，买机场的朋友多留意这 **5 个高危信号**：

- [x] **情况一**：大面积断网不修复（节点突然死了一大片，好几天都没人管）。
- [x] **情况二**：客服开始装死（之前秒回的客服，现在发工单、私信都没人理）。
- [x] **情况三**：电报群/频道摆烂（公告要么不更了，要么开始发些跟服务无关的鸡汤）。
- [x] **情况四**：网站抽风但收银台正常（官网隔三差五打不开，充值买套餐的页面比谁都流畅）。
- [x] **情况五**：老用户用不了，新用户还能买（你手里的订阅已经废了，网页上还在大肆招揽新客）。

> **稳妥玩法的“三不”原则**
